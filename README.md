# Smart-Ai-IoT-Dustbin
A smart waste management system using ESP8266 and Python Flask for fill-level monitoring and AI segregation.

This project is an IoT-based Smart Dustbin using:
- Raspberry Pi
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Load Cell 
- Machine Learning (trash classification)
- MQTT IoT connectivity

Features:
- Auto-open lid when a user comes near
- Detects fill level
- Detects weight of garbage
- Sends data to cloud 
- Classifies trash type
